# Validador de HTML em Java

Este projeto implementa um validador de HTML simples, que verifica se as tags estão corretamente abertas e fechadas, com suporte a interface gráfica.

# Funcionalidades:

- Interface gráfica com entrada de texto e botão de validação
- Verificação da estrutura de tags HTML
- Retorno de mensagens de erro caso a estrutura esteja incorreta
